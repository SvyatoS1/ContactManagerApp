# Contact Manager Application
Простий веб-додаток на базі ASP.NET Core MVC для керування контактами. Проєкт дозволяє імпортувати дані з CSV-файлів, зберігати їх у базі даних MS SQL та виконувати динамічне редагування, сортування та фільтрацію без перезавантаження сторінки.

## Основні можливості

**Імпорт CSV:** Завантаження та обробка файлів з автоматичним парсингом полів: Name, Date of Birth, Married, Phone, Salary.

**Inline Editing:** Редагування будь-якого поля контакту безпосередньо в таблиці.

**Динамічна фільтрація:** Пошук за будь-якою колонкою.

**Клієнтське сортування:** Можливість сортування даних за будь-яким атрибутом.

## Технологічний стек

**Backend:** .NET 10, ASP.NET Core MVC.

**ORM:** Entity Framework Core.

**Database:** Microsoft SQL Server.

**CSV Processing:** CsvHelper.

**Validation:** FluentValidation.

**Frontend:** JavaScript, HTML5 , CSS3 (Bootstrap 5).

## Встановлення та запуск

1. **Клонуйте репозиторій**:
```bash
https://github.com/SvyatoS1/ContactManagerApp.git

```

2. **Налаштуйте рядок підключення**:
У файлі `appsettings.json` змініть `DefaultConnection` на ваш локальний екземпляр SQL Server.

3. **Застосуйте міграції**:
```bash
dotnet ef database update

```

4. **Запустіть проєкт**:
```bash
dotnet run

```

## Вимоги до CSV-файлу

Для коректного імпорту контактів файл повинен відповідати наступним критеріям:

**Заголовки**: Файл повинен містити рядок заголовків з назвами: `Name`, `Date of Birth`, `Married`, `Phone`, `Salary`.

**Формат дати**: Дата народження повинна бути вказана у форматі **MM/DD/YYYY** (Місяць/День/Рік).

 **Типи даних**:
 
**Name**: Текстовий рядок.

**Married**: Логічне значення (`true`/`false` ).

**Phone**: Рядок з номером телефону.

**Salary**: Числове значення.
